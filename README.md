# tg_bot_image_transformer

## Описание

Данный проект является телеграмм ботом, основным фукнционалом которого является изменения стилевых составляющих
изображений, полученных от пользователя. 

Поддерживается следующий функционал
- перенос одного стиля на изображение;
- перенос одновременно двух стилей на изображение;
- перенос нескольких стилей по какой-то маске;
- какой-то предобученный GAN (ТУТ НУЖНО БУДЕТ НАПИСАТЬ БОЛЬШЕ КОНКРЕТИКИ);

В проекте используются 2 нейронные сети различных архитектур. Первая из них это NST (Neural Style Transfer), которая
отвечает за функционал, связанный за перенос стилей. Вторая GAN (Generative adversarial networks), данная сеть
отвечает за функционал создания уникального изображения.

Данный сервис работает в двух режимах
- Первый режим представляет собой телеграмм бота, который умеет выполнять основной функционал (описанный выше).
- Второй режим, это режим локально развёрнутого сервиса, который повторяет в точности функционал бота, однако 
его можно развернуть на любом локальном сервере, используя технологию Docker;

##  Подробное описание функционала

### Переносы стилей (TODO: Добавить картинки для наглядности работы)

В данном блоке опишем принцип работы, а также некоторые требования, связанные с переносом стиля. Любой из данных пунктов
функционала поддерживает любые изображения в формате jpg/jpeg, при этом вы можете загружать как ЧБ, так и Цветные 
изображения. Ниже опишем особенности работы каждого из пунктов:
- Переноc одного стиля на изображения: Для данного пункта вам необходимо загрузить 2 изображения, одно из которых 
   является контентным (т.е. изображением, на которое будет накладываться стиль), второе изображение будет являться 
   стилевым. Рекомендуется подбирать какие-то яркие стили, например Хохлома или Кубизм итд;
- Перенос одновременно двух стилей на изображение: данный пункт во многом повторяет предыдущий, с той лишь разницей,
что теперь вам необходимо загрузить 2 разных стиля, результирующим же изображением будет являться наложением 
  одновременно двух загруженных стилей;
- Перенос нескольких стилей по какой-то маске; Доступно будет 4 типа масок: 
  - Горизонтальная маска, т.е. исходное изображение делится горизонтально пополам и на каждую из половин переносится 
    свой стиль;
  - Вертикальная маска, аналогично пункту выше, только изображение пополам делится вертикально;
  - Диагональная маска, аналогично пункту выше, только изображение пополам делится по диагонали;
  - Сегментационаная маска, в качестве маски используется нейронная сеть, решающая задачу сегментации изображения, при 
    этом будет выделяется 2 класса: background (т.е. фон) и всё остальное;
